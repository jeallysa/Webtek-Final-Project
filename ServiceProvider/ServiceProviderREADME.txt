Starting NodeJs server:
1.)Open command prompt.
2.)Change the directory where the index.js is located. Index.js contains the javascript code for starting/ running the server.
3.)Type 'node index.js'. And the console will let you know if the connection is "Successful" with the database 
4.}Then the command will start the NodeJs server.
5.)At the browser, type the HostName and the Port to redirect you to the Dashboard (ex. http://localhost:81/dashboard).
6.)Now you are ready to see the comments, service requests, services and the create profile of the Service Provider.
